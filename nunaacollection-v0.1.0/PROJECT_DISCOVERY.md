# Project Discovery: Nunaa.Collection

## Project Vision

Nunaa.Collection คือเว็บไซต์สำหรับแบรนด์เสื้อผ้า everyday look ที่เน้นความสบาย ความเรียบง่าย และการแมตช์ได้ทุกคอลเลคชั่น โดยนำเสนอผ้าท้องถิ่นจากเชียงใหม่เป็นแกนหลักของแบรนด์

## Goals

- สร้างเว็บไซต์ที่สะท้อนภาพลักษณ์แบรนด์
- ใช้งานง่ายบนมือถือ
- โหลดเร็ว
- รองรับ custom domain
- ไม่มีค่า hosting รายเดือนในระยะเริ่มต้น
- พร้อมต่อยอดเป็น e-commerce เต็มรูปแบบ

## Target Audience

- ลูกค้าอายุประมาณ 20-40 ปี
- ชอบเสื้อผ้าเรียบง่าย ใส่ได้บ่อย
- สนใจผ้าท้องถิ่น งาน handmade และ slow fashion
- ใช้งานผ่านมือถือเป็นหลัก

## Brand Keywords

- Everyday Look
- Comfortable
- Minimal
- Local Fabric
- Chiang Mai
- Handmade
- Mix & Match

## Functional Requirements

- ดูสินค้า
- เพิ่มสินค้าลงตะกร้า
- ดูยอดรวม
- อ่านข้อมูลผ้า
- อ่านวิธีดูแลผ้า
- ดูขั้นตอนสั่งซื้อ
- ติดต่อร้านผ่าน Instagram

## Non-functional Requirements

- Responsive
- SEO friendly
- Fast loading
- Maintainable
- GitHub Pages compatible

## Initial Stack

- HTML
- CSS
- JavaScript
- GitHub Pages
- Domain: nunaacollection.com
